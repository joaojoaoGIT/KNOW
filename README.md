# Colégio Técnico KNOW – Portal Institucional

Portal institucional completo desenvolvido com **HTML5, CSS3 e JavaScript Vanilla**.

## Como usar

Abra o arquivo `index.html` em qualquer navegador moderno, ou sirva a pasta via servidor estático:

```bash
# Python
python3 -m http.server 8080

# Node
npx serve .
```

## Páginas

- `index.html` — Home institucional
- `sobre.html` — Sobre o colégio
- `cursos.html` — Listagem de cursos
- `curso-informatica.html` / `curso-enfermagem.html` / `curso-administracao.html` — Páginas individuais
- `estrutura.html` — Galeria de estrutura
- `matriculas.html` — Processo + formulário de leads
- `contato.html` — Formulário de contato
- `login.html` — Login e cadastro
- `area-aluno.html` — Plataforma EAD com vídeos
- `admin.html` — Painel administrativo
