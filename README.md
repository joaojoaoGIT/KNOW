# 🎓 KNOW – Portal Educacional

![KNOW Banner](images/logo-know.png)

> Plataforma educacional desenvolvida para apresentar cursos, informações institucionais e recursos interativos para alunos e visitantes.

---

# 📚 Sobre o Projeto

O **KNOW** é um portal educacional desenvolvido utilizando apenas **HTML, CSS e JavaScript**, oferecendo uma experiência moderna, intuitiva e responsiva para estudantes, visitantes e administradores.

O projeto foi criado com foco em aprendizado prático de desenvolvimento web, aplicando conceitos de estruturação de páginas, estilização avançada e programação com JavaScript.

---

# ✨ Funcionalidades

## 🌐 Área Institucional
- Página inicial responsiva
- Informações sobre a instituição
- Catálogo de cursos
- Apresentação da estrutura escolar
- Página de contato

## 📝 Matrículas
- Formulário de inscrição
- Cadastro de novos alunos
- Armazenamento de dados localmente

## 👨‍🎓 Área do Aluno
- Sistema de login
- Cadastro de usuários
- Acesso a conteúdos exclusivos

## 👨‍💼 Área Administrativa
- Painel administrativo
- Controle de informações cadastradas
- Gerenciamento de usuários

## 🎮 Recursos Interativos
- Quiz Vocacional
- Raspadinha Digital
- Componentes dinâmicos com JavaScript

# 🚀 Como Executar

1. Baixe ou clone o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo:

```plaintext
index.html
```

4. O sistema será aberto diretamente no navegador.

Não é necessário instalar programas ou bibliotecas adicionais.
